# IP Tracker with Telegram Notification

A simple IP tracker that sends notifications to a Telegram bot when an IP address is tracked.

## Features
* Tracks IP addresses and displays location information
* Sends notifications to a Telegram bot when an IP address is tracked
* Uses Leaflet.js for map display
* Uses ip-api.com for IP geolocation data

## Usage
1. Clone the repository and open `index.html` in a web browser
2. Enter an IP address in the input field and click "Track"
3. The IP address will be tracked and a notification will be sent to the Telegram bot

## Configuration
1. Replace `YOUR_BOT_TOKEN` with your Telegram bot token
2. Replace `YOUR_USER_ID` with your Telegram user ID

## Dependencies
* Leaflet.js
* ip-api.com
